# /assets — Wedding Media Files

Place the following files in this folder before deploying:

| File          | Description                                   |
|---------------|-----------------------------------------------|
| cover.jpg     | Hero / parallax background image              |
| couple.jpg    | Couple photo used in the "Our Story" section  |
| gallery1.jpg  | Gallery photo 1                               |
| gallery2.jpg  | Gallery photo 2 (wide span)                   |
| gallery3.jpg  | Gallery photo 3                               |
| gallery4.jpg  | Gallery photo 4                               |
| gallery5.jpg  | Gallery photo 5 (tall span)                   |
| gallery6.jpg  | Gallery photo 6                               |
| map.png       | Static map screenshot for the venue           |
| music.mp3     | Background music (autoplay disabled by default) |

> **Tip:** Optimise images before uploading (WebP or compressed JPEG, max ~1 MB each).
