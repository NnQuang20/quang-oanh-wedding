# /assets — Wedding Media Files

Place the following files in this folder before deploying:

| File          | Description                                   |
|---------------|-----------------------------------------------|
| cover.webp     | Hero / parallax background image              |
| couple.webp    | Couple photo used in the "Our Story" section  |
| gallery1.webp  | Gallery photo 1                               |
| gallery2.webp  | Gallery photo 2 (wide span)                   |
| gallery3.webp  | Gallery photo 3                               |
| gallery4.webp  | Gallery photo 4                               |
| gallery5.webp  | Gallery photo 5 (tall span)                   |
| gallery6.webp  | Gallery photo 6                               |
| map.png       | Static map screenshot for the venue           |
| music.mp3     | Background music (autoplay disabled by default) |

> **Tip:** Optimise images before uploading (WebP or compressed JPEG, max ~1 MB each).
